parent backward review current contract, fake-AI Cartesian payload coverage,
downstream progression blockers, no-second-review assertions, old-shape
rejection, and injected corrupt multi-gap hard blockers.

- **THEN** `missing_payload_cells` SHALL NOT include any parent backward review
  Cartesian cell
- **AND** router quality, packets, route, and terminal tier evidence SHALL own
  the parent backward review cells with current passing artifacts

#### Scenario: Release evidence cannot rely on progress-only background checks

  completed result artifacts
- **AND** progress-only background output SHALL NOT count as pass evidence
