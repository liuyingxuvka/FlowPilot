## Design

This change keeps the existing FlowPilot current-contract shape. It does not
add a runtime fallback path, legacy alias, or body-only substitute.

The control-plane Cartesian layer receives formal artifact faults from the
same registry-backed ContractExhaustionMesh source that already feeds the fake
AI responder. Each registry fault mode becomes a first-class mutation id in
the parent Cartesian alphabet. The parent matrix can then prove that bridge
rows are consumed by the current TestMesh owner instead of silently treating
the mutation family as unknown.

The liveness test fix is a test-alignment correction. The model and runner both
use `unknown_binding_evidence_marked_safe`; the test must assert that current
hazard name instead of the older label.
